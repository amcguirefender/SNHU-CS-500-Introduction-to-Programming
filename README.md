Introduction: This repository contains the work related to the SNHU Introduction to Programing course in the Python coding language. It is necessary to have access to Python to open the code files and Microsoft Visio to open the flowchart files. An outline of the documents and files included is as follows:
1)	02-CS 500 Module Two Journal: is a Word document outlining the use of conditional statements to create code relating to the purchase of stock from a brokerage account. Additionally, it contains a flowchart outlining the code process and an excerpt of the Python code to create the program.

2)	03-CS 500 Module Three Journal: is a Word document builds upon the scenario discussed in the CS 500 Module Two Journal with the addition of the use of functions and modular programming. It contains the code for the program without and with the use of functions for comparison purposes.

3)	04-CS 500 Module Four Journal: is a Word document outlining the use of list, tuples, and dictionaries in a payroll calculation application. Additionally, it contains an excerpt of the Python code to create the program and a sample user output calculation.

4)	05-CS 500 Module Six Journal: is a Word document discussing the use of object-oriented programming and creating an employee access control matrix based on the employee’s job role. Additionally, it contains an excerpt of the Python code to create the program and a sample of a general access control matrix.

5)	06-CS 500 7-2- Activity Files: is a ZIP file containing a process flowchart Visio file, the Python code file, and Word document outlining the creation of a simple calculator.  The Word document contains the process used to create the application as well as information regarding the testing performed on the code.

6)	07-CS 500 500 Project: is a ZIP file containing a process flowchart Visio file, the Python code file, a JSON file containing sample entry data, and Word document outlining the creation of the personal budget application. The Word document outlines the code’s purpose, the use of object-oriented programming and functions, and the testing performed on the code.

7)	08-CS 500 Module Ten Journal: is a Word document containing a reflection on the topics learned by completing the projects summarized above as well the future application of the skills gained.
